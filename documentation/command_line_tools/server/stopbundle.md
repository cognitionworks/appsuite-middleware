# NAME

stopbundle - stops a bundle

# SYNOPSIS

**stopbundle** [OPTIONS] bundlename


# DESCRIPTION

This command line tool stops a specific bundle.

# OPTIONS

**-h** *jmxHost*
: The JMX host

**-p** *jmxPort*
: The port

**-l** *jmxLogin*
: The optional login
 
**-pw** *jmxPassword*
: The optional password

# EXAMPLES

**stopbundle open-xchange-server**
