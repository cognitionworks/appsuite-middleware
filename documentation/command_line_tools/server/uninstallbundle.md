# NAME

uninstallbundle - uninstalls a bundle

# SYNOPSIS

**uninstallbundle** [OPTIONS] location


# DESCRIPTION

This command line tool uninstall a specific bundle.

# OPTIONS

**-h** *jmxHost*
: The JMX host

**-p** *jmxPort*
: The port

**-l** *jmxLogin*
: The optional login
 
**-pw** *jmxPassword*
: The optional password

# EXAMPLES

**uninstallbundle open-xchange-server**
