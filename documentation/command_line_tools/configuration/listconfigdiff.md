# NAME

listconfigdiff - lists all reloadable options.

# SYNOPSIS

**listconfigdiff** [-f <filename>] | [-h]

# DESCRIPTION

Lists the differences between the default and the installed configuration.

# OPTIONS

**-h**, **--help**
: Prints a help text

**-f** *fileName*
: The file name to list the difference for.
