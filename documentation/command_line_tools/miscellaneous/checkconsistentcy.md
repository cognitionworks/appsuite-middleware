# NAME

checkconsistency - checks for config mistakes

# SYNOPSIS

**checkconsistency** [-h]

# DESCRIPTION

This command line tool checks the middleware's configuration for some well known configuration mistakes.

# OPTIONS

**-h**, **--help**
: Prints a help text.
